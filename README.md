# oasi-social-app
O Oasi é uma rede social projetada para oferecer um espaço digital mais leve, humano e saudável. Diferente das plataformas tradicionais, a Oasi foca em bem-estar, intencionalidade, silêncio digital e relações mais autênticas, reduzindo ruído, métricas tóxicas e estímulos excessivos.
